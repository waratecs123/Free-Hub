# This project is **100% static and private by design**:

## What We Don't Do:
- **Collect any data** from visitors
- **Require registration** or user accounts
- **Use tracking** or analytics services
- **Run backend servers** or databases
- **Store personal information**
- **Use cookies** or browser storage
- **Connect to external APIs** for user functionality

## What This Is:
- **Static HTML/CSS/JS files** only
- **Reference content** that you can use offline
- **Client-side code** that runs only in your browser
- **No dependencies** to install or maintain
- **Completely transparent** codebase

## Your Privacy is Guaranteed:
- Browse the catalog **anonymously**
- Clone/fork **without providing any personal information**
- Use locally **without internet connection** after loading
- No data leaves your device **ever**
- All filtering and search happens **in your browser**

**This is just a website. Like a digital catalog, but completely private.**
